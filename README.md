# EasyWeather

EasyWeather is a responsive and fast weather application built with **HTML**, **CSS**, and **JavaScript**, using the free **Open-Meteo API**.  
It displays **current**, **past 7 days**, and **16-day forecast** weather data for any location worldwide.

## Features
- Live location search with suggestions  
- Past and future weather data (hourly + daily)  
- Temperature and precipitation charts  
- Clean, responsive UI  
- No API key required (Open-Meteo)
